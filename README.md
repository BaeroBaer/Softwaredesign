## Ausfuehrung (Abgabe)

Einmal im Projektordner ausfuehren:

```bash
python setup.py
```

Das reicht aus, um alles noetige einzurichten (Pakete installieren + Check).

Optional:

- Alle Notebooks ausfuehren:

```bash
python setup.py --run-notebooks
```

- Danach direkt Jupyter starten:

```bash
python setup.py --open-notebook
```
